Materi Dasar C++

C++/Cpp adalah bahasa pemrograman yang merupakan pengembangan dari bahasa C. C++ mendukung pemrograman berorientasi objek, serta fitur-fitur lain seperti pengelolaan memori yang lebih baik dan pemrograman generik. Ini menjadikannya lebih fleksibel dan kuat dalam membangun berbagai jenis aplikasi, dari perangkat lunak sistem hingga aplikasi desktop dan game.

1. Struktur Dasar Program C++

#include <iostream>
using namespace std;
int main() {
      //kode program
    return 0;
}

#include <iostream> berfungsi untuk menyertakan header input/output standar.

int main() { ... }: Tempat kerja/eksekusi suatu kode C++.

2. Output 

cout << "hasil akhir muncul" <<endl;

cout: Digunakan untuk menampilkan output ke konsol/tampilan akhir.

<<: Operator Shift Left untuk mengirim/memindahkan data ke cout agar bisa ditampilkan pada konsol.

3. Input dari Pengguna

int angka;
cin >> angka;

cin: Digunakan untuk menerima input/data dari pengguna.

>>: Operator Shift Right untuk membaca data dari cin.

4. Variabel dan Tipe Data

int hutang = 250;
double setengah = 0,5;
char huruf = 'A';
string kata = "contohnya";
bool benar = true;
bool salah = false;

int: Tipe data untuk bilangan bulat.

double: Tipe data untuk bilangan desimal.

char: Tipe data untuk 1 karakter.

string: Tipe data untuk lebih dari 1 karakter.

bool: Tipe data untuk 2 nilai saja yaitu true atau false.

5. Operator Aritmatika

int angka1 = 10;
int angka2 = 5;
int tambah = angka1 + angka2;
int kurang = angka1 - angka2;
int perkalian = angka1 * angka2;
int pembagian = angka1 / angka2;
int mod = angka1 % angka2;

+ - * / %: Operator untuk operasi aritmatika dasar.

6. Struktur Kontrol

Percabangan (If /else)

if (bayar < 18.000) {
    cout << "belum lunasss" << endl;
} else {
    cout << "oke maksi" << endl;
}

Pengulangan For

for (int i = 0; i < 5; ++i) {
    cout << i << endl;
}

Pengulangan While

int merah = 0;
while (merah < 5) {
    cout << merah << endl;
    ++merah;
}

7. Array

int isi[5] = {1, 2, 3, 4, 5};

int isi[5]: Mendeklarasikan array dengan lima elemen bertipe int.
