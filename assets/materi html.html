Pengenalan HTML yang Mendetail

Halaman ini menjelaskan dasar-dasar HTML dengan penjelasan rinci dan contoh kode.

1. Struktur Dasar HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. Struktur dasar HTML melibatkan elemen-elemen berikut:


<!DOCTYPE html> 
<html lang="id"> 
    <head>
        <meta charset="UTF-8"> 
        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
        <title>Judul Halaman</title> 
    </head>
    <body>
        <header>
            <h1>Ini adalah Heading 1</h1> 
        </header>
        <section>
            <p>Ini adalah paragraf pertama.</p> 
            <p>Ini adalah paragraf kedua.</p> 
        </section>
    </body>
</html>

2. Elemen dan Tag HTML

HTML menggunakan berbagai tag untuk menyusun dan menampilkan konten. Berikut adalah beberapa tag dasar:

  • <h1> hingga <h6>: Heading dari yang paling penting hingga yang kurang penting.
  • <p>: Tag untuk paragraf.
  • <a>: Tag untuk membuat tautan. Contoh: Kunjungi Contoh.
  • <img>: Tag untuk menyisipkan gambar. Contoh:
  • <ul> dan <ol>: Tag untuk membuat daftar tidak berurutan (unordered list) dan berurutan (ordered list). <li> digunakan untuk item dalam daftar.

Contoh HTML untuk daftar dan gambar:


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<img src="gambar.jpg" alt="Deskripsi Gambar" width="200">

3. Atribut HTML

Tag HTML sering memiliki atribut untuk memberikan informasi tambahan. Atribut ditulis di dalam tag pembuka. Berikut adalah beberapa contoh:


<a href="https://www.example.com" target="_blank">Kunjungi Contoh</a>


<img src="gambar.jpg" alt="Deskripsi Gambar" width="200">

4. Format Teks

Tag format teks memungkinkan Anda untuk memodifikasi tampilan teks di halaman web. Beberapa tag format teks yang umum:

  • <b>: Menebalkan teks. Contoh: teks tebal.
  • <i>: Miringkan teks. Contoh: teks miring.
  • <strong>: Menandai teks penting (biasanya menebalkan). Contoh: teks penting.
  • <em>: Menandai teks yang diutamakan (biasanya miring). Contoh: teks diutamakan.

5. Tabel dan Formulir

HTML memungkinkan pembuatan tabel dan formulir untuk mengorganisir data dan mengambil input dari pengguna:

Tabel


<table border="1">
    <tr> 
        <th>Header 1</th> 
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td> 
        <td>Data 2</td>
    </tr>
</table>

Formulir


<form action="/submit" method="post">
    <label for="name">Nama:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Kirim">
</form>

6. Komentar

Komentar dalam HTML digunakan untuk menambahkan catatan dalam kode. Mereka tidak ditampilkan di browser:


<!-- Ini adalah komentar -->

Gunakan komentar untuk menjelaskan bagian-bagian kode atau menandai bagian yang perlu diperhatikan.
