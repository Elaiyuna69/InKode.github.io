    Pengenalan Python

    Halaman ini memberikan pengantar dasar tentang bahasa pemrograman Python.

    1. Apa itu Python?

    Python adalah bahasa pemrograman tingkat tinggi yang mudah dipelajari dan digunakan. Dikenal karena sintaksisnya yang bersih dan kode yang mudah dibaca, Python digunakan dalam berbagai bidang seperti pengembangan web, analisis data, kecerdasan buatan, dan banyak lagi.

    2. Struktur Dasar Python

    Berikut adalah contoh program Python sederhana yang mencetak teks ke layar:

            # Program Python pertama
print("Hello, World!")

    Penjelasan:

    • #: Digunakan untuk menulis komentar.
    • print(): Fungsi bawaan untuk mencetak output ke layar.

    3. Tipe Data

    Python memiliki beberapa tipe data bawaan:

    • int: Tipe data untuk bilangan bulat. Contoh: 5.
    • float: Tipe data untuk bilangan desimal. Contoh: 3.14.
    • str: Tipe data untuk string (teks). Contoh: "Hello".
    • list: Tipe data untuk daftar yang dapat diubah. Contoh: [1, 2, 3].
    • tuple: Tipe data untuk urutan yang tidak dapat diubah. Contoh: (1, 2, 3).
    • dict: Tipe data untuk kamus (key-value pairs). Contoh: {'key': 'value'}.
    • bool: Tipe data untuk nilai Boolean. Contoh: True atau False.

    Contoh mendeklarasikan berbagai tipe data:

            # Tipe data
number = 5 # int
pi = 3.14 # float
message = "Hello, Python!" # str
numbers_list = [1, 2, 3] # list
coordinates = (10.0, 20.0) # tuple
person = {'name': 'Alice', 'age': 25} # dict
is_active = True # bool

    4. Operator

    Operator digunakan untuk melakukan operasi pada variabel dan nilai. Beberapa operator dasar:

    • + : Penjumlahan. Contoh: 5 + 3.
    • - : Pengurangan. Contoh: 5 - 3.
    • * : Perkalian. Contoh: 5 * 3.
    • / : Pembagian. Contoh: 5 / 2.
    • % : Modulus (sisa bagi). Contoh: 5 % 2.
    • // : Pembagian bulat. Contoh: 5 // 2.
    • ** : Pangkat. Contoh: 2 ** 3.

    Contoh penggunaan operator:

            a = 10
b = 3
print(a + b) # Penjumlahan
print(a - b) # Pengurangan
print(a * b) # Perkalian
print(a / b) # Pembagian
print(a % b) # Modulus
print(a // b) # Pembagian bulat
print(a ** b) # Pangkat

    5. Kontrol Alur

    Python menyediakan struktur kontrol alur untuk mengontrol eksekusi kode:

    • if, elif, else: Untuk percabangan. Contoh:
      •         x = 10
if x > 5:
print("x lebih besar dari 5")
elif x == 5:
print("x sama dengan 5")
else:
print("x kurang dari 5")
    • for: Untuk perulangan. Contoh:
      •         for i in range(5):
print(i)
    • while: Untuk perulangan dengan kondisi. Contoh:
      •         count = 0
while count < 5:
print(count)
count += 1

    6. Fungsi

    Fungsi adalah blok kode yang dapat digunakan kembali. Berikut adalah contoh mendefinisikan dan memanggil fungsi:

            def greet(name):
return f"Hello, {name}!"
        
print(greet("Alice"))

    Penjelasan:

    • def: Digunakan untuk mendefinisikan fungsi.
    • return: Mengembalikan nilai dari fungsi.

    7. Modul

    Modul adalah file Python yang berisi fungsi dan variabel yang dapat digunakan di skrip lain. Berikut adalah contoh menggunakan modul math:

            import math

print(math.sqrt(16)) # Menghitung akar kuadrat dari 16
print(math.pi) # Menampilkan nilai pi

    Penjelasan:

    • import: Digunakan untuk mengimpor modul.
    • math.sqrt(): Menghitung akar kuadrat.
    • math.pi: Menampilkan nilai pi.
